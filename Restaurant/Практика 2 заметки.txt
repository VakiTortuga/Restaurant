# требования меню:
- задание свойств конструируемого объекта (сделать заказ)
- вывод свойств объекта (просмотреть заказ)
- выполнение методов объекта (приготовить заказ)

# требования китчен:
- содержит список (какой? меню или/и заказов?)
- выводит [список меню]

# как организовать меню

по идее меню с блюдами для заказа должно быть 100%
но операции (методы) блюд нужно проводить не над элементами меню, а над заказами
необходим ли отдельный класс для заказа (номер заказа + тип пиццы)

Необходимо создать:
- [список меню] (добавление элемента в список заказов)
- [очередь заказов] (где можно отражать процесс готовки)
- метод вывода пользовательского меню

========================================

использовать везде свойство Weight

можно ли добавлять просто food в меню?

стоит вынести создание нового элемента меню отдельным пунктом

в ShowFoodMenu нужно измнеить вывод блюд покрасивше

Структура (пункты) меню:

В1
1. ~Добавить заказ~
	Выводит [список меню] и вариант [кастомной] пиццы
	Добавляет в [список заказов]
2. ~Просмотреть [список заказов]~
	Выводит [список заказов] с базовой инфой
	При вводе номера заказа выдает его характеристику
3. ~Готовка [заказа]~
	Выводит [список заказов] с базовой инфой
	При вводе номера заказа выдает доступные действия
	При выборе действия срабатывает соответствующий метод
4. ~Выход~
