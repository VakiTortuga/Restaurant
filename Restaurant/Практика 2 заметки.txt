# требования меню:
- задание свойств конструируемого объекта (сделать заказ)
- вывод свойств объекта (просмотреть заказ)
- выполнение методов объекта (приготовить заказ)

# требования китчен:
- содержит список (какой? меню или/и заказов?)
- выводит [список меню]

Структура (пункты) меню:

В1
1. ~Добавить заказ~
	Выводит [список меню] и вариант [кастомной] пиццы
	Добавляет в [список заказов]
2. ~Просмотреть [список заказов]~
	Выводит [список заказов] с базовой инфой
	При вводе номера заказа выдает его характеристику
3. ~Готовка [заказа]~
	Выводит [список заказов] с базовой инфой
	При вводе номера заказа выдает доступные действия
	При выборе действия срабатывает соответствующий метод
4. ~Выход~
